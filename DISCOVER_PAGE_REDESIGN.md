### 2. 内容流卡片设计

#### 卡片头部 - 用户信息
```
┌─────────────────────────────────┐
│ [头像] 用户名        [关注按钮]   │
│        5小时前·上海              │
└─────────────────────────────────┘
```

**用户头像**:
- 圆形，半径20px
- 网络图片加载

**用户名**:
- 字体大小: 13px
- 字重: bold
- 颜色: 深灰色 (#0F172A)

**时间和位置**:
- 字体大小: 11px
- 颜色: 浅灰色 (#94A3B8)
- 格式: "Xhours ago·Location"

**关注按钮**:
- 背景: 透明
- 边框: 紫色 (#905FFF)，1.5px
- 文字: 紫色
- 字体大小: 12px
- 圆角: 16px
- 内边距: 水平12px，垂直6px

#### 卡片内容 - 分类标签
```
[🏋️ 健身达人]
```

**分类标签**:
- 背景: 浅紫色 (#F3E8FF)
- 文字: 紫色 (#905FFF)
- 字体大小: 11px
- 字重: w600
- 圆角: 12px
- 内边距: 水平10px，垂直4px

#### 卡片内容 - 图片
```
┌─────────────────────────────────┐
│                                 │
│         [内容图片]               │
│                                 │
│                                 │
└─────────────────────────────────┘
```

**图片样式**:
- 宽度: 100%（减去padding）
- 高度: 280px
- 圆角: 12px
- 加载失败: 显示占位符

#### 卡片底部 - 互动区域
```
❤️  💬  ➡️        🔖
1,250次点赞
健身达人Leo 今天的腹肌感受太强了！配合了拉伸的进阶
训练，感受力量提升升升升。💪坚持运动 #拉动App
3条评论
```

**互动按钮**:
- 点赞: 心形图标
- 评论: 气泡图标
- 分享: 箭头图标
- 收藏: 书签图标
- 大小: 18px
- 颜色: 灰色 (#475569)

**点赞数**:
- 字体大小: 11px
- 字重: w600
- 颜色: 灰色 (#475569)
- 格式: "X次点赞"

**内容标题**:
- 字体大小: 12px
- 字重: bold
- 颜色: 深灰色
- 行高: 1.4

**内容描述**:
- 字体大小: 11px
- 颜色: 灰色 (#475569)
- 行数: 最多3行
- 行高: 1.4

**评论数**:
- 字体大小: 10px
- 颜色: 浅灰色 (#94A3B8)
- 格式: "X条评论"
